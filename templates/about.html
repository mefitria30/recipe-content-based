ABOUT

Aplikasi ini dibuat dengan tujuan pengembangan ilmu pengetahuan di bidang machine learning dan Progressive Web App (PWA).

Sebagai pengembang, saya mengucapkan terima kasih kepada penyedia template SpiceHaven yang telah menghadirkan desain halaman web indah dan membantu mempercepat proses pengembangan aplikasi ini.

Fokus utama saya adalah membangun sistem rekomendasi yang akurat dari sisi data science, sekaligus menghadirkan pengalaman pengguna yang maksimal melalui fitur progressive web apps. Seluruh pengembangan ini merupakan bagian dari tugas akhir jenjang S1 di perguruan tinggi.
